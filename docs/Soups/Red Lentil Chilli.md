
# Red Lentil Chilli

## :salt: Ingredients
| How much  | What |
| -------- |--------------------- |
| **1**        | large white onion     |
| **2**        | medium carrots        |
| **2**        | garlic gloves         |
| **1 tsp**    | cumin                 |
| **0.25 tsp** | chilli powder         |
| | oregano               |
| | pepper                |
| | cayenne pepper        |
| **1 can**    | whole canned tomatoes |
| **200 ml**   | water                 |
| | pepper                |
| | salt                  |
| **160 g**    | red lentils           |




## :pencil: Instructions
Dice 1 large white onion. Also chop 2 medium carrots into small pieces. Sate the
onion until translucent and add the carrots. Mince 2 garlic gloves and add to the onions and carrots.
Season with a 1 tsp cumin and 0.25 tsp chilli powder until fragrant. Feel free to add more spices such as oregano,
pepper or cayenne pepper.

Add 1 can whole canned tomatoes and 200 ml water. Bring to a simmer and cook until tomatoes are soft.

Season well with pepper and salt before adding 160 g red lentils. Simmer until lentils are soft (15 min).
