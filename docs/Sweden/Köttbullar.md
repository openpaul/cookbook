
# Köttbullar

## :salt: Ingredients
| How much  | What |
| ----- |----------- |
| **250 g** | minced pork |
| **250 g** | minced beef |
| **1**     | egg         |
| | salt        |
| | pepper      |
| | paprica     |
| **50 g**  | bread crums |




## :pencil: Instructions
Mix 250 g minced pork with 250 g minced beef. Add 1 egg, salt, pepper, paprica and 50 g bread crums.
Gently kneet it until firm consitency.
Form 20 meatballs of 60 g each.

Fry on low-medium high lighly oiled pan until golden brown.

**Tip**: Use lid on pan to prevent early browning and ensure they are cooked trough.

## Mashed Potatoes

### :salt: Ingredients
| How much  | What |
| ------ |-------- |
| **1 kg**   | Potatoes |
| **80 g**   | butter   |
| **200 ml** | Milk     |
| | salt     |
| | nutmeg   |

### :pencil: Instructions
Boil 1 kg Potatoes until very soft.
Peel the potatoes while warm and add 80 g butter and 200 ml Milk.
Add salt and nutmeg and then stomp until consitency has reached smooth/chunky.

**Extra Fluffy**: Use a potatoe ricer to mash the potatoes and then mix the other ingredients gentily in.

## Swedish Cucumber Salad

### :salt: Ingredients
| How much  | What |
| ------ |-------------- |
| **1**      | cucumber       |
| | white vinnegar |
| **1 tbsp** | sugar          |
| | dill           |
| | salt           |
| | pepper         |

### :pencil: Instructions
Peel 1 cucumber and slice thinly into rings.
Add salt and after 5 min press liquid out.

For the dressing mix white vinnegar, 1 tbsp sugar and dill. Season with salt and pepper.

Dress the salad and let rest until serving time. At least 20 min.
